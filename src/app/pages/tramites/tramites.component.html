Requisitos para Gestiones Municipales

Para obtener el carnet de conducir, se deben cumplir con los siguientes requisitos:

  • Fotocopia de DNI.
  • Certificado de domicilio.
  • Constancia de grupo sanguíneo.

Para organizar eventos bailables, se deben cumplir con los siguientes requisitos:

  • Solicitar el permiso con 15 días de antelación.
  • Certificado de seguridad del lugar.
  • Contar con personal de seguridad adecuado.
  • Asegurar que el evento sea apto para todo público.

Para obtener habilitación comercial, se deben presentar los siguientes documentos:

  • Inscripción de la actividad comercial.
  • Comprobante de pago de impuestos.
  • Certificado de sanidad e higiene del establecimiento.

Para la inscripción de títulos (automotor o catastro), se requiere:

  • Presentación del título original.
  • Copia del título de propiedad.
  • Pago de tasa administrativa.
  • Comprobante de identificación del propietario.
